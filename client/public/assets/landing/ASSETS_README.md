# GlutiSafe Landing Assets

Generated landing-page visual assets for `public/assets/landing`.

## Backgrounds
- `backgrounds/organic-lines.svg`: subtle flowing decorative lines.
- `backgrounds/soft-green-blobs.svg`: airy mint blob shapes for hero layering.
- `backgrounds/floating-particles.svg`: light floating particles for motion/background use.
- `backgrounds/hero-bg-shape.svg`: layered abstract hero shape for phone/content backdrops.
