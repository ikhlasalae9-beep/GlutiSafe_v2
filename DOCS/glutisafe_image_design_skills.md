# GlutiSafe — Image & Design Creation Skills

## 1. Objectif du fichier

Ce fichier sert comme guide pratique pour créer des images, posts, stories, highlight covers, logos, mockups et supports marketing pour la marque **GlutiSafe**.

Il doit être utilisé comme référence avant toute création visuelle afin de garder une identité cohérente, professionnelle et reconnaissable.

## 3. Palette de couleurs

### Couleurs principales

| Rôle | Couleur | Code |
|---|---|---|
| Background principal | Blanc | `#FFFFFF` |
| Background alternatif | Gris très clair chaud | `#F7F8F6` |
| Accent principal | Vert GlutiSafe | `#008F45` |
| Background foncé | Vert foncé premium | `#004B3A` |
| Texte principal | Dark Charcoal | `#1D252B` |
| Accent doux | Mint Green | `#A8CFA5` |

### Règles

- Utiliser 3 à 4 couleurs maximum par visuel.
- Utiliser le vert uniquement pour les mots importants, icônes et CTA.
- Utiliser le blanc pour les contenus éducatifs.
- Utiliser le vert foncé pour les teasers, posts de lancement et supports plus forts.

À utiliser pour :

- Explication des services

Caractéristiques :

À utiliser pour :

Caractéristiques :

- fond vert foncé
- texte blanc
- accents mint green

## 5. Formats recommandés

### Instagram Post

- Portrait : `1080 x 1350 px`
- Ratio : `4:5`
- Usage : post principal, teaser, annonce, campagne marketing

### Instagram Square

- Carré : `1080 x 1080 px`
- Ratio : `1:1`
- Usage : profile picture, covers, posts simples

### Instagram Story

- Story : `1080 x 1920 px`
- Ratio : `9:16`
- Usage : stories, highlights, guides rapides, FAQ

### Highlight Cover

- Carré : `1080 x 1080 px`
- Ratio : `1:1`
- Usage : covers des highlights

---

## 6. Règles de composition

### Toujours faire

- garder beaucoup d’espace vide
- rendre le titre grand et lisible
- utiliser une hiérarchie claire
- mettre les mots importants en vert
- garder le logo propre et visible
- aligner les éléments proprement
- utiliser des icônes simples
- garder le design respirant
- vérifier que le texte est lisible sur mobile

### Éviter

- trop de texte
- trop de couleurs
- fond trop chargé
- logo coupé
- texte collé aux bords
- fausses interfaces illisibles
- style enfantin
- style médical trop dramatique
- promesses médicales exagérées
- visuels trop sombres pour les contenus éducatifs

---

## 7. Logo usage

### Profile picture

Utiliser l’icône seule du logo, centrée, sur fond blanc, sans texte long.

### Posts

Utiliser le logo complet si l’espace le permet.

### Stories

Logo petit et propre en haut.

### Règles

- ne pas déformer le logo
- ne pas changer les couleurs principales
- ne pas ajouter d’effets 3D lourds
- garder une marge autour du logo
- ne pas coller le logo aux bords

## 10. Copywriting visuel

### Ton

Le texte doit être :

- clair
- court
- rassurant
- professionnel
- positif
- accessible
- simple à comprendre

### Phrases clés

- Scannez. Vérifiez. Choisissez.
- Moins de doute, plus de clarté.
- Votre guide gluten-free arrive bientôt.
- Des choix sans gluten plus sûrs.
- GlutiSafe arrive pour vous accompagner.
- Restez connectés.
- Safe Gluten-Free Dining.

### À éviter

- “guérit”
- “traitement”
- “solution médicale”
- “100% garanti”
- “aucun risque”
- “diagnostic”
- tout claim médical non vérifié

## 11. Workflow de création d’image

### Étape 1 — Choisir le format

Déterminer le type de contenu :

- Post 4:5
- Story 9:16
- Cover 1:1
- Profile picture 1:1
- Banner
- Contact card

### Étape 2 — Choisir le thème

- White Theme pour information, éducation, services, highlights
- Dark Green Theme pour teaser, lancement, announcement

### Étape 3 — Définir le message

Un seul message principal par design.

Exemples :

- Bientôt disponible
- Scannez vos produits
- Vérifiez avant de consommer
- C’est quoi le gluten ?
- Comment se protéger ?

### Étape 4 — Construire le prompt

Le prompt doit toujours contenir :

- marque : GlutiSafe
- format exact
- style visuel
- couleurs
- texte exact
- composition
- éléments à éviter
- consigne de lisibilité

### Étape 5 — Vérification finale

Avant validation, vérifier :

- le logo n’est pas coupé
- le texte est correct
- le format est respecté
- les couleurs sont cohérentes
- l’image n’est pas surchargée
- les visages ou mains ne sont pas déformés
- les infos de contact ne sont pas coupées
- aucune faute en français
